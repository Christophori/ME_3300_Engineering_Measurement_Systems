\section{Introduction}
Measurement of strain is a fundamental tool used by engineers in both research and design. Strain measurements are often used in systems where force or pressure measurements are required. In some applications the strain (and associated stress) of a part can only be determined by measurement, as calculation is either too complicated or not accurate enough. In this experiment, you will learn to use a metallic strain gauge to measure strain in an end-loaded cantilevered beam. Simultaneously, you will calculate the stress due to bending caused by the end load. You will use these values to calculate a stress value. Finally, your stress and strain plot will be used to determine the modulus of elasticity (or Young’s modulus) of the beam. \\

\end{itemize}

\subsection{Part 2: Preparing for analysis of the Strain and Stress Data.}
\begin{enumerate}
    \item Measure the beam using the following tools/dimensions:
    \begin{itemize}
        \item Use a tape measure to measure the distance between the center of the strain gauge and the point of loading, L. Note the resolution of the tape measure in meters.
        \item Use a vernier caliper to measure the width of the beam, w. Note the resolution of the vernier caliper in meters.
        \item Use a micrometer to measure the beam thickness, t. Note the resolution of the micrometer in meters.
    \end{itemize} 
    \item Determine local gravity using this \href{https://www.sensorsone.com/local-gravity-calculator/}{local gravity website}  (use the Latitude value of 46.7324 and the elevation of 2579 ft).
    \item From solid mechanics we get the relationship $\sigma = My/I$. You will use this relationship for stress calculations. The stress equation, including all the measured variables, is provided in equation 4.
    \begin{eqnarray}
    \sigma &=& \frac{M y}{I}, \\
      &=& \frac{P L (t/2)}{w t^3 / 12}, \\
      &=& \frac{6PL}{wt^3}, \textrm{and} \\
      &=& \frac{6mgL}{wt^2}.
    \end{eqnarray}
    \item The measured strain, $\varepsilon $, may be found from the half-bridge Wheatstone  circuit
    \begin{eqnarray}
    \frac{\delta V_o}{V_i  G} &=& \frac{GF \varepsilon }{2},\\
    \varepsilon  &=& 2 \frac{\delta V_o}{V_i G} \times \frac{1}{GF}
    \end{eqnarray}
    where $V_i$ is excitation voltage, $GF$ = 2.012 is the gauge factor, $G$ is the instrumentation amplifier gain, and $\delta V_o$ is the change in output voltage from loaded to unloaded condition. 
\end{enumerate}

    \end{itemize}
    \item Set up data acquisition for single channels of DAQ system.
    \begin{itemize}
        \item ``AI0": output from the instrumentation amplifier
    \end{itemize}
    \begin{enumerate}[label=\alph*.]
        \item Running your program:
        \begin{itemize}
            \item Save a data file using this naming convention: \boldpurp{Mass\#\#\_@@.csv}
            \item Change \#\# to the current mass in kilogram.
            \item Change @@ to either loaded or unloaded
            \begin{itemize}
                \item Loaded scenarios have the weight included during the measurements.
                \item Unloaded at measured before and after loaded scenarios when the weight is removed.
            \end{itemize}
        \end{itemize}
    \end{enumerate}
    \item Record the average voltage value in the table provided (Table \ref{tab:data}).
    \item Method of measuring strain data for each weight, example for 0.4 kg:
    \begin{enumerate}
        \item Record voltage measurement for unloaded scenario($[V_{i,1}]_{avg}$.)
        \item Load 0.4 kg weight, record voltage measurement for loaded scenario ($[V_{i,2}]_{avg}$.)
        \item Record a second voltage measurement for unloaded scenario ($[V_{i,3}]_{avg}$.)
        \begin{itemize}
            \item Your 2nd column should have two unloaded voltage values.
        \end{itemize}
        \item Subtract the unloaded from loaded: $G\delta V_{o,1}=[V_{i,2}]_{avg} -[V_{i,1}]_{avg}$ and $G\delta V_{o,2}=[V_{i,2}]_{avg} -[V_{i,3}]_{avg}$.
        \begin{itemize}
            \item Your 3rd column should have two $G\delta V_o$ values.
        \end{itemize}
        \item Average the two values: $[G\delta V_o]_{avg}=\frac{G\delta V_{o,1}+G\delta V_{o,2}}{2}$
    \end{enumerate}
    \item Repeat the process for the remaining weights.
    \item NOTE: Make sure \textbf{not to touch} the wires in your setup since the gain of 500 will make it sensitive to any changes. 
    \item NOTE: $[G \delta V_o]_{avg}$ should not be modified. Since $V_o$ is amplified and $G$ is included for clarity. In equation \ref{fig:ErrEq} $V_o$ is expressed without $G$ to ensure the user does not include $G$ in the error propagation $V_o$ but $V_o$ should still remain unchanged.
\end{enumerate}

\subsection{Part 6: Compare  uncertainty analysis from two approaches}
\begin{itemize}
\item To estimate uncertainty in Young’s Modulus, you will use two approaches and compare results
\begin{enumerate}
\item Propagation of error
Use the equation  \ref{fig:ErrEq}.
\begin{equation}
E = \frac{3mgLV_{i}G (GF)}{wt^2 \delta V_{o}}.
\label{fig:ErrEq}
\end{equation}
\item Error from curve fit. The standard error of the slope can be found according to:
\begin{equation}
s_{a1} = s_{yx} \sqrt{ \frac{1}{\Sigma_{i=1}^{N} (x_i - \overline{x})^2}},
\end{equation}
where $s_{yx}$ is the standard error of the fit.
\end{enumerate}
\end{itemize}
