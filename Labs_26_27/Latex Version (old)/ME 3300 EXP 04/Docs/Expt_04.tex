\section{Introduction}
In this experiment, you will compare the measured signals from multiple sources including: an oscilloscope, a digital multimeter, a table-top digital multi-meter, and the NI-MyDAQ data acquisition device to view both DC and AC signals through a voltage divider circuit. 

A typical AC signal is shown in figure \ref{fig:demoSignal}. The DC offset component is the offset of the reference signal (that is, zero volts) and $V_{pp}$ is the difference between the maximum and minimum voltage in the AC signal. Therefore; in the provided example, the offset and $V_{pp}$ would be 2 volts and 6 volts, respectively.

\subsection{Part 5: Measure dynamic signals with two different methods.}
\begin{enumerate}
    \item Dial in the following AC signal on the function generator: $y(t) =2.5+ 2sin(2 \pi 0.5 t)$
    \item Collect the single-channel voltage output from (``AI1").
    \begin{itemize}
        \item Duration: 5 times the time period of the function $y(t)$
        \begin{itemize}
            \item \textbf{Do not sample below 0.2 seconds.}
        \end{itemize}
        \item Sample rate: 1000 samples per second.
        \begin{itemize}
            \item \textbf{NOTE: this sample rate will remain constant but the frequency in function generator will be varied.}
        \end{itemize}
    \end{itemize}
    \item Use the zoom X-axis button on the \textbf{oscilloscope} to zoom until 2-5 waves are visible. (You may have to modify this for every new signal)
    \item \textbf{Practice changing the function generator}: increase the input frequency, the frequency of the function generator $y(t)$, incrementally starting from 0.5 Hz to 1,125 Hz.

    \item \textbf{For your post-lab}, save the acquired data for $y(t)$ at the following five frequencies (relative to the Nyquist frequency $f_N$):
        \begin{itemize}
            \item $0.2 f_N$: Well below the Nyquist limit. The signal should be sampled accurately without distortion.
            \begin{itemize}
                \item Note: find $f_N$ based on 1000 Hz and this will remain constant for the remaining frequencies.
            \end{itemize}
            
            \item $0.9 f_N$: Still below Nyquist, but close enough that the sampled signal will look ``blocky'' or discretized due to limited time resolution.
            
            \item $1.0 f_N$: At the Nyquist frequency. The signal may appear distorted or ambiguous. This is the critical point where aliasing begins.
            
            \item $1.10 f_N$: Slightly above Nyquist. Determine both the predicted alias frequency (using the folding formula) and the actual observed alias frequency.
            
            \item $2.25 f_N$: Well above Nyquist. Again, determine both the predicted alias frequency and the actual observed alias frequency.
        \end{itemize}
    
    \item Show your results to the TA to confirm you have collected all required data for the post-lab assignment.     
    \item Modify the provided sample code: \boldpurp{MATLAB\_Dynamic\_plot.m}.
\end{enumerate}

\subsection{Oscilloscope}
There are two oscilloscope brands that will be used in the lab: the Agilent and the Hewlett Packard oscilloscopes. They have the same general functionality but the buttons are in different places. The following are basic instructions for each.
\begin{figure}[!hb]
\centering
\includegraphics[width=5.6in]{TektronixOscilloscope02.png}
\caption{Oscilloscope keypad layout}
\end{figure}
To setup for a measurement, turn the appropriate channel(s) on (if only using one channel, turn the other off).  Press the corresponding Channel Button to access the options for that channel. The channel is on when the button is green.  Press the button once to turn it on, press it again to turn it off.
The Function Buttons change their function based on the current operation. Their current function is displayed on the side of the screen next to the corresponding button. For example, when you press one of the Channel Buttons, the function buttons will change to the options available for that particular channel.\\

To manually set the scale the display, use the corresponding {\bf Voltage Scale} and the {\bf Time Scale} adjustments. They are labeled Volts/div and Time/div respectively. The voltage is displayed on the y-axis and the time is displayed on the x-axis. The current voltage scale is listed in the top left corner of the screen, and the current time scale is listed in the top center of the screen. The display area is split into a grid for both axes. The number displayed for each scale corresponds to the amount of voltage or time represented by each grid space. Adjust each scale to a value that makes sense for the signal being measured (remember that frequency = 1/time). The voltage scale must be set for each channel independently. \\

The {\bf Ground Shift} adjustment controls where the incoming signal ground is referenced to. It is labeled Position, and is adjusted independently for each channel. This acts as a voltage offset for the display, and allows a signal to be viewed even if that signal has voltages that would be out of the range of the display when ground is referenced to 0V. For most measurements, start by bringing the ground to 0V. A number on the side of the screen indicates where the ground currently is. \\

The {\bf Time Shift} does the same thing for time that the ground shift does for voltage. For most measurements, bring it to 0.  A number at the top of the screen indicates the current position. \\

For more display options, press the Main/Delayed button and adjust the settings using the Function Buttons (see function button note). The Main option (selected by default) keeps the signal stationary and traces its path across the screen. The Roll option keeps the trace stationary and scrolls the signal across the screen. Sometimes it is easier to use Roll to verify that the scope is still updating, when Main seems to be unchanging.
